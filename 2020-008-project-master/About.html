What is Multisplit?

Multisplit was developed by a team of four developers. It is a web application that is used to split bills in a shared household. The purpose of Multisplit is to make it easier for household members to keep track of which household member payed which house expense or did not pay. In addition to this, it helps balance the household expenses fairly. A registered member can add a new house or join an already existing house of choice. Each registered member can post any expense they have paid for whether payed using Multisplit or payment has taken outside Multisplit such as cash or EFT. Each registered member is able to access a log from the Multisplit that captures all actions taken by each member of the same household.
